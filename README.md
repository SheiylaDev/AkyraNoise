# Akyra Noise

Application Windows desktop pour FiveM (et autres apps voix) qui capture ton micro,
applique une suppression de bruit RNNoise temps reel, et renvoie le flux propre
vers un cable audio virtuel utilisable comme entree micro.

## Telechargement

Derniere version : [Releases](https://github.com/SheiylaDev/AkyraNoise/releases/latest)

Telecharge `AkyraNoise.exe`, lance-le. La configuration de VB-Cable est entierement
automatique au premier lancement (necessite les droits administrateur).

## Mise a jour

Les mises a jour sont detectees et installees automatiquement au demarrage de
l'application.
